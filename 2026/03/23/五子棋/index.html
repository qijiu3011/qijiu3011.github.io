五子棋游戏控制台与easyX版本

简介

使用 C++ 实现简单的的双人五子棋游戏，棋盘大小为 15×15。玩家轮流输入坐标进行落子，程序能够实时显示棋盘状态，并自动判断胜负和平局。

其中：

  • X 表示黑棋；
  • O 表示白棋；
  • 任意方向连续五个棋子即可获胜。

功能

1. 棋盘显示

使用二维数组 b[15][15] 存储棋盘状态，通过 print() 函数输出当前棋盘。

2. 落子与玩家切换

通过 check() 函数判断输入位置是否合法，并完成落子。利用变量 nowPl 记录当前玩家，每回合结束后切换到另一方继续游戏。

3. 胜负判断

checkWin() 函数以当前落子位置为中心，对横向、纵向以及两条对角线共四个方向进行搜索，当连续棋子数达到 5 时判定当前玩家获胜。

4. 平局判断

变量 cnt 用于统计落子总数，当棋盘被下满且无人获胜时，判定为平局。

主要函数

clear() 清屏
print() 输出棋盘
check() 判断落子是否合法
checkWin() 判断是否形成五连子
nowPlayer() 切换玩家
main() 控制整个游戏流程

图形界面版本

利用 EasyX 图形库实现了窗口版五子棋，通过鼠标完成落子，并实时绘制棋盘和棋子。程序支持胜负判断、平局判断、非法落子提示以及按 R 键重新开始游戏，提高了游戏的交互性和可视化效果。
在控制台版本的基础上，通过加入图形界面和鼠标事件处理，实现了更加直观的五子棋游戏，加深了对图形编程和事件驱动思想的理解。
