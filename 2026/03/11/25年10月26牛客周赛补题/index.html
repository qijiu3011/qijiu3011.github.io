C - 小苯的真假游戏

问题描述

有 n 个位置围成一个环，每个位置 i 给出一个值

$$
s_i \in {0,1}
$$

表示它对左边位置 (i-1) 的状态作出的声明。

每个位置自身也有一个状态

$$
x_i \in {0,1}
$$

要求所有位置满足：

  • 若 $x_i = 1$，则声明 正确
  • 若 $x_i = 0$，则声明 错误

其中 1 号位置的左边是 n 号位置。

考察知识点

  • 异或
  • 环形约束
  • 逻辑关系转化

题解思路

根据题意可以推出每个位置之间存在如下关系：

  • 当 $s_i = 1$ 时，两人的状态相同
  • 当 $s_i = 0$ 时，两人的状态不同

将所有约束沿环展开，可以发现问题只与所有 $s_i$ 的 异或和 以及 n 的奇偶性 有关。

最终结论为：

  • 若 n 为偶数且 s_i 的异或和为 0
  • 或 n 为奇数且 s_i 的异或和为 1

则存在 2 种合法方案。

否则 无解。

D - 小苯的区间选数2.0

问题描述

给定 n 个区间，需要从每个区间中选择一个数字，构成一个长度为 n 的序列，使得序列的 MEX 最大化。

求 MEX 的 最大值。

考察知识点

  • 贪心
  • 优先队列
  • 区间调度

题解思路

为了使 MEX 最大化，我们希望尽可能依次构造：

假设当前需要构造数字 x：

  • 所有满足 l ≤ x 的区间都可以用于选择 x
  • 在这些区间中，应优先选择 右端点最小的区间

这样可以尽可能保留更大的区间供后续使用。

实现方法：

  1. 按 l 对区间排序
  2. 使用优先队列维护所有 l ≤ x 的区间的 r
  3. 每次取 r 最小的区间

若当前没有区间可以覆盖 x，则当前 x 即为最大 MEX。
