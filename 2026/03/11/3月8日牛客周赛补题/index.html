C - Unique Number

问题描述

对给定序列 c[n]（初始所有值为 0）进行如下操作：

  • 选择一个前缀 [1, i] (1 ≤ i ≤ n)
  • 将前缀中所有元素的值增加 1

同时，你还会得到一个 限制序列 d[n]。

要求在操作过程中 任意时刻

$$
c_i \le d_i
$$

最终能使序列 c 中 不同数值的种类数最多是多少？

考察知识点

  • 贪心
  • 构造
  • 单调序列

题解思路

要使 c 中不同数值种类数最大，最终构造的数组应该是单调递减的。因为每次操作只能增加前缀，越靠后的位置被增加的次数越少，所以最终值满足 $c_1$ ≥ $c_2$ ≥ … ≥ $c_n$。

构造策略如下：

  1. 从第一个位置开始，取最大可能值 $c_1$ = $d_1$
  2. 对于第 i 个位置，考虑前一个值 $c_{i-1}$：
    • 如果 $c_{i-1}$ - 1 ≤ $d_i$，则可以取 $c_i$ = $c_{i-1}$ - 1
    • 如果 $c_{i-1}$ - 1 > $d_i$，则只能取 $c_i$ = $d_i$
  3. 每确定一个值，种类数 +1
  4. 当构造到 $c_i$ = 0 时，后面所有位置都只能是 0，此时退出循环

不需要真正构造出整个数组，只需要模拟这个过程计数即可。
