ACM 对拍器 —— 带 GUI 的数据生成与自动对比工具

发表于 2026-05-18

项目简介

写 ACM 题时，经常需要验证优化程序是否正确。传统做法是手写对拍脚本（.bat 或 .py），但对于不太熟悉脚本的同学不太友好。

这个项目用 Win32 GUI 写了一个带界面的对拍器，集成了数据生成和自动对比功能，点点鼠标就能跑对拍。

功能概览

1. 数据生成器（8 种类型）

类型 说明 参数示例
随机整数数组 生成 n 个 [a, b] 范围内的整数 n=100 a=-1000 b=1000
随机排列 1~n 的随机排列 n=1000
随机字符串 可指定字符集的字符串 len=100 chars=oO
随机区间 n 个 [l, r] 区间对 n=100 a=1 b=1000
随机树 n 个节点的无根树（边列表） n=1000
随机图 n 点 m 边的连通图 n=100 m=500
随机平面点 n 个 (x, y) 坐标点 n=100 a=-10 b=10 c=-10 d=10
随机数对 n 个 (a, b) 数对 n=100 a=1 b=100 c=1 d=100

2. 对拍器核心

  • 自动运行暴力程序（brute.exe）和优化程序（fast.exe）
  • 逐 token 比较输出，精确到第几个 token 不同
  • 发现差异时自动保存输入数据到 diff_时间戳.txt
  • 支持实时停止、计数统计

3. GUI 界面

纯 Win32 API 实现，无任何外部依赖：

  • 数据生成参数面板
  • 程序路径选择（带浏览按钮）
  • 彩色运行结果列表
  • 状态栏实时反馈

使用方法

  1. 准备好暴力程序 brute.exe 和优化程序 fast.exe（读 stdin，写 stdout）
  2. 双击 acm_judge.exe 打开 GUI
  3. 选择数据类型，填写参数
  4. 点「生成数据」预览一组数据
  5. 点「开始对拍」自动跑 N 组随机数据并对比
