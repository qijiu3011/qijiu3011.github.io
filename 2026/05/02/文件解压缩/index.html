基于哈夫曼树的文件压缩与解压软件

发表于 2026-05-02

项目简介

本项目基于哈夫曼树实现文件压缩与解压功能。程序通过统计字符出现频率建立哈夫曼树，生成对应编码，并采用二进制方式存储压缩数据，提高存储效率。同时，将字符频率信息一并保存到压缩文件中，使得解压时无需额外编码表即可恢复原文件内容。

支持功能：

  • 文件压缩
  • 文件解压
  • Windows 文件选择对话框
  • 单文件 .huf 压缩格式
  • 二进制位压缩
  • 控制台菜单操作

哈夫曼树原理

哈夫曼编码是一种带权路径长度最短的前缀编码方法。

基本过程：

  1. 统计字符出现频率；
  2. 建立哈夫曼树；
  3. 深度优先遍历生成编码；
  4. 使用编码替换原字符；
  5. 按位存储编码结果；
  6. 解压时重建哈夫曼树并恢复原文件。
