## 题目链接

[ABC 432 C - Candy Tribulation](https://atcoder.jp/contests/abc432/tasks/abc432_c)

## 问题描述

对 n 个孩子分发大糖果和小糖果：
- 小糖果 x 克
- 大糖果 y 克

对于 a[i]，代表第 i 个孩子需要分配的糖果个数。要求所有孩子分配到的糖果总重相等。

## 错因分析

公式推出来了，但是不会写检查总重相等那部分的代码。不过这 C 题怎么比之前的难这么多。。。

## 考察知识点

数论（同余定理）

## 题解思路

设每个孩子最终得到的总重量为 w，则：

$$
w = x \cdot a[i] + k[i] \cdot (y - x)
$$

其中 k[i] 是给第 i 个孩子的大糖果数量（相对于全部分小糖果的增量）。

### 1. 确定 w 的范围

- **最小值**：所有孩子都尽可能用小糖果时，取最大值：
  $$
  w_{\min} = \max_{i} (x \cdot a[i])
  $$

- **最大值**：所有孩子都尽可能用大糖果时，取最小值：
  $$
  w_{\max} = \min_{i} (y \cdot a[i])
  $$

如果 $w_{\min} > w_{\max}$，则无解，输出 -1。

### 2. 同余条件

由 $w = x \cdot a[i] + k[i] \cdot (y - x)$ 可得：

$$
x \cdot a[i] \equiv w \pmod{(y-x)}
$$

因此，所有 $x \cdot a[i]$ 对 $(y-x)$ 取模必须相等，否则无解。

### 3. 计算总糖果数

取 $w = w_{\max}$（或任意可行值），则：

$$
k[i] = \frac{w - x \cdot a[i]}{y - x}
$$

总糖果数（即大糖果的总增加量）为：

$$
\sum_{i=1}^{n} k[i]
$$

## 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
using ll = long long;
const ll N = 2e5+10;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    ll n, x, y;
    cin >> n >> x >> y;
    ll a[n];
    for(ll i = 0; i < n; i++) cin >> a[i];
    
    // 最小值：取 x*a[i] 的最大值
    ll minsum = -1;
    for(ll i = 0; i < n; i++) 
        minsum = max(minsum, x * a[i]);
    
    // 最大值：取 y*a[i] 的最小值
    ll maxsum = 1e18;
    for(ll i = 0; i < n; i++) 
        maxsum = min(maxsum, y * a[i]);
    
    if(minsum > maxsum)
    {
        cout << -1;
        return 0;
    }
    
    // 检查同余条件：所有 x*a[i] % (y-x) 必须相等
    ll ys = (x * a[0]) % (y - x);
    if(ys < 0) ys += (y - x);
    for(ll i = 1; i < n; i++) 
    {
        ll aidys = (x * a[i]) % (y - x);
        if(aidys < 0) aidys += (y - x);
        if(aidys != ys)
        {
            cout << -1;
            return 0;
        }
    }
    
    // 计算总糖果数
    ll sum = 0;
    for(ll i = 0; i < n; i++)
    {
        if((maxsum - x * a[i]) % (y - x) != 0)
        {
            cout << -1;
            return 0;
        }
        ll temp = (maxsum - x * a[i]) / (y - x);
        sum += temp;
    }
    
    cout << sum;
    return 0;
}
