在进行思考前，我们应先明确vector在STL中是什么分类。容器分为序列式和关联式，vector就是序列式的一种，与之同为序列式的有array、list、slist、deque等，实现较关联式容器的红黑树map等是更容易的。

正文：根据vector的源码，我复刻了一个简易版vector的大体框架

vector需要表示用户操作的当前数据的起始地址，结束地址，还需要真正的最大地址。所以共3个迭代器分别指向数据的头start，数据的尾finish，数组的尾endofstorage。由于只是一个初始的简易复现，对于尾使用了m_len表示数组长度，进行间接表示。

构造函数相关

在源码中，vector的初始化块进行了严谨的异常机制，由于知识过多以后再进行实现。

vector是一种class template，是在栈上分配的，所以我们不需要手动释放内存，生命周期结束后自动调用析构从而释放调用空间。但这不代表进程完全不进行delete操作，如上文框架中的m_Data指向的内存是在堆上通过 new[] 分配的，编译器不会自动释放，必须在 ~MyVector() 中写 delete[] m_Data，否则这块堆内存永远不会被释放，造成内存泄漏。

虽然不需要手动释放内存，但直接调用析构函数释放内存的操作是合法的

insert

在vector插入元素分为三种情况

插入点后的现有元素多于新增元素个数

从pos点插入n个元素，先构造出start + pos - n大小空间，调整长度，将插入点 后的元素copy到备用空间，最后从插入元素位置填充新值

插入点后的现有元素少于等于新增元素个数

从pos点插入n个元素，先判断是否有备用空间，在插入点当前end()指针后构造一个元素，将原先插入点后的元素copy到备用空间，最后从插入点填充新值

如果备用空间不足

先配置新空间，再分段copy元素，填充插入值，最后调整迭代器位置

其中需要注意迭代器失效问题，因为插入元素可能使容器元素整体迁移导致存放原容器元素的空间不再有效，或是因为删除元素，使某些元素次序发生变化导致原本指向某元素的迭代器不再指向期望指向的元素。

题目链接

ABC 432 C - Candy Tribulation

问题描述

对 n 个孩子分发大糖果和小糖果：

  • 小糖果 x 克
  • 大糖果 y 克

对于 a[i]，代表第 i 个孩子需要分配的糖果个数。要求所有孩子分配到的糖果总重相等。

错因分析

公式推出来了，但是不会写检查总重相等那部分的代码。不过这 C 题怎么比之前的难这么多。。。

考察知识点

数论（同余定理）

题解思路

设每个孩子最终得到的总重量为 w，则：

$$
w = x \cdot a[i] + k[i] \cdot (y - x)
$$

其中 k[i] 是给第 i 个孩子的大糖果数量（相对于全部分小糖果的增量）。

1. 确定 w 的范围

  • 最小值：所有孩子都尽可能用小糖果时，取最大值：
    $$
    w_{\min} = \max_{i} (x \cdot a[i])
    $$

  • 最大值：所有孩子都尽可能用大糖果时，取最小值：
    $$
    w_{\max} = \min_{i} (y \cdot a[i])
    $$

如果 $w_{\min} > w_{\max}$，则无解，输出 -1。

2. 同余条件

由 $w = x \cdot a[i] + k[i] \cdot (y - x)$ 可得：

$$
x \cdot a[i] \equiv w \pmod{(y-x)}
$$

因此，所有 $x \cdot a[i]$ 对 $(y-x)$ 取模必须相等，否则无解。

3. 计算总糖果数

取 $w = w_{\max}$（或任意可行值），则：

$$
k[i] = \frac{w - x \cdot a[i]}{y - x}
$$

总糖果数（即大糖果的总增加量）为：

$$
\sum_{i=1}^{n} k[i]
$$

代码实现

#include<bits/stdc++.h>
using namespace std;
using ll = long long;
const ll N = 2e5+10;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    ll n, x, y;
    cin >> n >> x >> y;
    ll a[n];
    for(ll i = 0; i < n; i++) cin >> a[i];
    
    // 最小值：取 x*a[i] 的最大值
    ll minsum = -1;
    for(ll i = 0; i < n; i++) 
        minsum = max(minsum, x * a[i]);
    
    // 最大值：取 y*a[i] 的最小值
    ll maxsum = 1e18;
    for(ll i = 0; i < n; i++) 
        maxsum = min(maxsum, y * a[i]);
    
    if(minsum > maxsum)
    {
        cout << -1;
        return 0;
    }
    
    // 检查同余条件：所有 x*a[i] % (y-x) 必须相等
    ll ys = (x * a[0]) % (y - x);
    if(ys < 0) ys += (y - x);
    for(ll i = 1; i < n; i++) 
    {
        ll aidys = (x * a[i]) % (y - x);
        if(aidys < 0) aidys += (y - x);
        if(aidys != ys)
        {
            cout << -1;
            return 0;
        }
    }
    
    // 计算总糖果数
    ll sum = 0;
    for(ll i = 0; i < n; i++)
    {
        if((maxsum - x * a[i]) % (y - x) != 0)
        {
            cout << -1;
            return 0;
        }
        ll temp = (maxsum - x * a[i]) / (y - x);
        sum += temp;
    }
    
    cout << sum;
    return 0;
}
